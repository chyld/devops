# Environment Variables

## How to create a variable for the current process only

- `X=3`
- `DOG=fido`

## How to create a variable for the current and all sub-processes

- `export Z=1`
- `export MONTH=august`

## How to access environment variables

- `echo $X`
- `echo $Z`
- `echo $DOG`
- `echo $MONTH`
